# Baby Raising App
ドットインストールの「Reactで作るTodo管理アプリ」を参考に学習しています。
あくまで、自己学習用です。
ここから育児記録機能を追加して育児記録とタスク管理が両方できるアプリを目指します。

## 出典
このプロジェクトは、[ドットインストール](https://dotinstall.com/)の「Reactで作るTodo管理アプリ」を参考に個人の学習目的で作成しています。
